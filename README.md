# 表格比对工具

> v2.0.0

逐列比对两个 Excel 文件，差异标红。

## 使用方法

1. 双击 `表格比对工具.bat`
2. 设置比对选项（自动记住配置）
3. 选择原始文件和对比文件
4. 选择工作表（单表自动跳过）
5. 选择要比对的列（支持全选/全不选）
6. 如开启关键列匹配，选择用于匹配行的列
7. 比对完成后弹窗查看统计，选择是否打开结果文件

## 功能

### 比对选项
- **忽略大小写** — A 与 a 视为相同
- **忽略日期格式差异** — `2020/1/1` 与 `2020-01-01` 视为相同
- **数值容差** — 如 0.01 以内视为相同
- **排除列模式** — 列多时选不需比对的列，其余自动比对
- **关键列匹配行** — 按指定列值（如 ID）匹配，不依赖行号顺序
- **自定义表头行** — 0=自动检测，其他数字=指定行号
- **差异汇总 Sheet** — 可选是否生成差异明细表
- **记住配置** — 设置自动保存到 config.json

### 比对能力
- 自动检测表头行（前 10 行中非空单元格最多的行）
- 列选择器支持「全选」「全不选」
- 按列名自动配对比对
- 空单元格跳过不标红
- 全空行自动跳过
- 新增的行标绿，缺少的行标黄
- 差异单元格标红加粗白字

## 环境要求

- Windows 10+
- Excel 或 WPS
