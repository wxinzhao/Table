# 表格比对工具

> v5.0.0（最终版本）

逐列比对两个 Excel 文件，差异标红。专为 MAC / IP 地址比对优化。

## 使用方法

1. 双击 `表格比对工具.bat`
2. 选择原始文件和对比文件
3. 选择工作表（支持 Ctrl 多选 / 全选）
4. 自动配对同名 Sheet，不同名可手动配对
5. 选择要比对的列（支持全选/全不选）
6. 表头不同时弹出手动列配对界面
7. 比对完成后选择：打开结果文件 / 关闭

## 功能

### 自动识别（始终生效）
- **MAC 地址** — 自动识别 `AA:BB:CC`、`AA-BB-CC`、`AABB.CCDD`、`AABBCC` 等格式，统一后比对
- **IP 地址** — 自动去除前导零，`192.168.001.001` = `192.168.1.1`
- **忽略大小写** — 始终自动忽略

### 比对能力
- 自动检测表头行（前 10 行中非空单元格最多的行）
- 列选择器支持「全选」「全不选」
- 按列名自动配对比对，表头不同时支持手动列配对
- 空单元格跳过不标红
- 全空行自动跳过
- 新增的行标绿，缺少的行标黄
- 差异单元格标红加粗白字

### 导出
- 结果文件命名：`文件名_比对结果_yyyyMMdd_HHmmss.xlsx`
- 副本表自动新增「说明」列，标注每行差异原因

- Windows 10+
- Excel 或 WPS
